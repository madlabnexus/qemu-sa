# Machine Configuration Files

Pre-configured machine profiles for QEMU-SA. These will be implemented in Phase 4.

See [docs/08-MACHINE-PROFILES.md](../../docs/08-MACHINE-PROFILES.md) for specifications.

Planned profiles:
- `dos-gaming.cfg` — DOS gaming PC (486DX2-66, SB16+OPL3, Cirrus VGA)
- `win311.cfg` — Windows 3.11 PC (Pentium, SB16+OPL3, Cirrus VGA)
- `win98-3dfx.cfg` — Windows 98 + 3Dfx (KVM, AC97, qemu-3dfx)
- `winxp-vfio.cfg` — Windows XP + VFIO (KVM, HDA, GPU passthrough)
