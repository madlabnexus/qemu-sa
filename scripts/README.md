# Scripts

Utility scripts for QEMU-SA development and deployment.

Planned:
- `build.sh` — build QEMU-SA with all features
- `test-dos.sh` — quick-launch DOS gaming profile
- `test-win98.sh` — quick-launch Win98 3Dfx profile
- `check-vfio.sh` — verify VFIO/IOMMU setup
- `check-iommu-groups.sh` — list IOMMU groups and devices
